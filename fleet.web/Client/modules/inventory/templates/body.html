Fleet is a demo / playground application designed to highlight a number of technologies. For more information, see Fleet on github: github.com/johndmathis/fleet

What's interesting about this, so far?

  • RequireJs is used to load the entire application.
  • The header, this body, and footer sections are all defined in two Marionette modules, called 'main' and 'inventory'.
  • The module definitions span multiple files, which are all loaded by RequireJS.
  • The templates are in separate files as well, and are loaded asynchronously once the inventory module is ready.
  • The inventory module is loaded on-demand, reducing initial load time.
